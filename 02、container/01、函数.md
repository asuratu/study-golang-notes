## 函数（一等公民）
```
func div(a, b int) (q, r int) {
	q, r = a/b, a%b
	return
}
```
1. 函数的返回值可以去多个名字
2. 仅用于非常简单的函数
3. 对于调用者而言没有区别

### 可变参数列表
```
func sum(numbers ...int) int {
    fmt.Printf("numbers 的类型：%T\n", numbers)
    sum := 0
    for _, v := range numbers {
        sum += v
    }
    return sum
}
```

### 函数语法回顾
1. 返回值类型写在最后面
2. 可返回多个值
3. 函数作为参数
4. 函数没有默认参数、可选参数
